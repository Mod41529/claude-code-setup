최신 GitHub 트렌드 리포트를 읽고 브리핑한다. 다음 순서대로 실행해라:

1. `~/projects/agent-orchestration/reports/` 에서 가장 최근 `github-trends-*.md` 파일을 찾아 읽어라.

2. 아래 형식으로 출력해라:

---

**GitHub 트렌드 브리핑 — [날짜]**

**즉시적용 항목**
| 레포 | 이유 | 적용 포인트 |
|---|---|---|

**참고 항목**
- 레포 + 이유 한 줄

---

3. 즉시적용 항목 중 **지금 바로 적용하면 좋을 TOP 3** 추천:
   - 현재 오케스트레이션 시스템(`~/projects/agent-orchestration/`)에 통합 가능한 것 우선
   - 설치/설정 복잡도가 낮은 것 우선
   - 실제 워크플로우 개선 효과가 큰 것 우선

   각 추천 항목: 왜 지금 좋은지 (2-3문장) + 적용 방법 간략 설명 + 예상 소요 시간

4. 마지막에 물어봐라: "어떤 항목 적용할까요?"
   선택하면 바로 적용 작업 시작.
