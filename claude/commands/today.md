오늘의 브리핑을 시작한다. 다음 파일들을 읽어라:
- ~/projects/agent-orchestration/SCHEDULE.md
- ~/projects/agent-orchestration/SOMEDAY.md
- ~/projects/agent-orchestration/RECURRING.md
- ~/projects/agent-orchestration/session.md (있으면 — 마지막 세션 요약 확인용)

아래 형식으로 출력해라:

---

**[오늘 날짜, 요일]**

**마감 박박**
- `## 마감 있음 (Deadline)` 섹션 내 항목 전체 출력
- 날짜 태그(`03-13`, `03-24` 등) 파싱 → D-day 계산 후 앞에 표시
  - D-0 또는 초과: 🔴 D-0 (또는 D+n)
  - D-1 ~ D-3: 🟠 D-n
  - D-4 ~ D-7: 🟡 D-n
  - D-7 초과 또는 날짜 없음: 날짜 그대로

**오늘 (Today)**
- SCHEDULE.md `## 오늘 (Today)` 섹션 내 항목 출력
- 비어 있거나 완료 항목만이면 "아직 설정 안 됨" 출력

**오늘의 반복 항목**
- RECURRING.md에서 오늘 요일에 해당하는 항목 출력

**다른 세션 진행 중 - [/]**
- SCHEDULE.md에서 `- [/]` 항목 전체 출력 (추천 포커스에서 제외)

**추천 포커스 (3개)**
- `## 언제든 (Anytime)` 섹션에서만 선정 (마감 있음·Today 섹션 제외)
- `- [/]` 항목은 추천하지 않음 (이미 다른 세션에서 진행 중)
- 선정 기준 (우선순위대로 적용):
  1. 오늘이 평일이면 #회사/#개발 우선, 주말이면 #라이프 #크리에이티브 우선
  2. [높] > [중] > [낮]
  3. 목록 위쪽(위에 있는) 항목 우선
- 각 항목 아래 이유 한 줄씩

**Someday 힐끗보기**
- SOMEDAY.md에서 카테고리별로 1개씩, 총 3개만 랜덤하게 꺼내서 보여줘
- 형식: `[카테고리] 항목명` + 짧은 코멘트
- 목적: 아이디어가 묻히지 않도록 가끔 눈에 띄게

---

마지막에 한 마디: 오늘 컨디션이나 방향에 대한 짧은 코멘트 (1-2문장).
